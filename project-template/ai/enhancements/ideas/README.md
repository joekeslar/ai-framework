# Ideas Parking Lot

Drop ideas here that aren't ready to plan yet.

## How to use

1. Copy `ai-framework/templates/idea.md`
2. Name it descriptively: `feature-name.md` (no number prefix needed)
3. Fill it in and leave it here until ready
4. When ready to build: copy it into `ai/enhancements/NNN-name/plan.md`, strike the
   idea through in `ENHANCEMENTS.md`, and **delete the file from here** — once promoted,
   it's tracked as an enhancement. Keeping this folder to only-still-unplanned ideas is
   what makes a glance (or `../status.sh`) honest.

## Example

```
ideas/
├── social-sharing.md
├── barcode-scanner.md
└── meal-planning.md
```
